## 🚀 クイックスタート

### 新しいメニューを作成する場合

1. **準備**
   ```bash
   # カード情報を確認
   python check_card_usage.py
   ```

2. **ガイド確認**
   - `MENU_CREATION_GUIDE.md`を開いて手順を確認

3. **テンプレート使用**
   - `MENU_TEMPLATE.md`をコピーして、メニュー情報を記入

4. **カード選定**
   - `CARD_REFERENCE.md`を参照してカードを選定
   - `card_name.csv`で詳細を確認

5. **データ投入**
   - Pythonスクリプトを作成するか、EXCELに直接入力

6. **チェック**
   - `MENU_CREATION_GUIDE.md`のチェックリストで確認

## ⚠️ 重要な注意事項

### カード番号について
- **必ず`card_name.csv`のカードIDを使用すること**
- カード番号は1〜44の範囲内
- 鑑定文はカードの意味（キーワード、概要）に沿って作成

### データ整合性
- メニューID、項目ID、結果IDの命名規則を守る
- メニュー、小項目、結果の件数が整合していること

## 🆘 トラブルシューティング

### よくある問題

**Q: カード番号がおかしいと言われた**
→ `card_name.csv`でカードIDを確認（1〜44）

**Q: どのカードを選べばいいか分からない**
→ `CARD_REFERENCE.md`でカテゴリー別に確認

**Q: 鑑定文が書けない**
→ `card_name.csv`の「detail」列を読んでカードの意味を理解

**Q: 既存メニューを参考にしたい**
→ `check_card_usage.py`で既存のカード使用パターンを確認
